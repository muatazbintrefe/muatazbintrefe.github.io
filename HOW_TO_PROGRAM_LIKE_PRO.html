C++ Programming Basics

Learn C++ programming from basic includes to object-oriented programming with classes. This guide will walk you through creating a complete Car class with practical examples.

Complete C++ Example: Car Class

Full Program Code

// Include necessary libraries #include <iostream> #include <string> using namespace std;
// Define the Car class class Car { private: string brand; string model; int year; double speed;
public: // Constructor Car(string b, string m, int y) { brand = b; model = m; year = y; speed = 0; }
// Method to increase speed void accelerate() { speed += 10; cout << brand << " " << model << " is accelerating. Speed: " << speed << " km/h" << endl; }
// Method to decrease speed void brake() { if (speed > 0) { speed -= 10; cout << "Braking... Speed: " << speed << " km/h" << endl; } }
// Method to display car information void displayInfo() { cout << year << " " << brand << " " << model << endl; cout << "Current speed: " << speed << " km/h" << endl; } };
// Main function - program entry point int main() { // Create car objects Car myCar("Toyota", "Camry", 2023); Car sportsCar("Ford", "Mustang", 2024);
// Use the objects myCar.displayInfo(); myCar.accelerate(); myCar.accelerate(); myCar.brake();
cout << "\n";
sportsCar.displayInfo(); sportsCar.accelerate();
return 0; }

Code Explanation

Lines 1-3: Include necessary libraries and use standard namespace

Lines 5-10: Define the Car class with private member variables

Lines 12-18: Constructor that initializes the car object

Lines 20-24: accelerate() method increases speed by 10

Lines 26-32: brake() method decreases speed by 10

Lines 34-38: displayInfo() method shows car details

Lines 41-55: main() function creates objects and tests methods
